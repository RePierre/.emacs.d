* JavaScript development
** Install =rjsx-mode=
   #+begin_src emacs-lisp :results silent
     (use-package rjsx-mode
       :ensure t
       :mode "\\.js\\'")
   #+end_src
** Intall =prettier-js=
   Requires =sudo apt install prettier= or =npm install -g prettier=.
   #+begin_src emacs-lisp :results silent
     (use-package prettier-js
       :ensure t
       :after (rjsx-mode)
       :hook (rjsx-mode . prettier-js-mode))
   #+end_src
